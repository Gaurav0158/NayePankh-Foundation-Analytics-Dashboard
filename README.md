# NayePankh-Foundation-Analytics-Dashboard
This project demonstrates how Data Analytics can support NayePankh Foundation in monitoring volunteer participation, donation trends, internship applications, and social media engagement.  A Power BI dashboard was developed using a simulated dataset because real organizational data was not publicly available.

This project aims to demonstrate how an analytics dashboard can help NayePankh Foundation:

Monitor volunteer participation
Track donation performance
Evaluate campaign effectiveness
Measure internship engagement
Analyze social media lead generation
Support strategic decision-making
📊 Dashboard Features
1. Key Performance Indicators (KPIs)

The dashboard provides an overview of:

Total Volunteers
Total Donations
Total Internship Applications
Total Social Media Leads
Total Campaigns
2. Donation Analysis

The dashboard includes donation trend visualizations that help identify:

Monthly fundraising patterns
High-performing periods
Campaign contribution analysis

This can help the organization optimize fundraising strategies.

3. Volunteer Analytics

Volunteer-related insights include:

Total volunteer participation
City-wise volunteer distribution
Campaign-wise volunteer involvement

These insights can support volunteer recruitment and engagement efforts.

4. Campaign Performance Analysis

The dashboard compares campaign performance across multiple initiatives, helping identify:

Most active campaigns
Campaign participation levels
Campaign donation contributions

This information can be used to improve future campaigns.

5. Social Media Analytics

Social media lead generation is tracked to evaluate:

Audience engagement
Awareness campaign effectiveness
Lead generation performance
6. Geographic Analysis

Map-based visualizations provide insights into volunteer distribution across different cities.

This enables the organization to identify:

High-engagement regions
Areas requiring outreach
Potential expansion opportunities
